What is 3DOM Classify?

3DOM Classify Viewer is an in-browser tool built on BabylonJS that lets you load point clouds or models (ply, las, glb), visually inspect and label them, train a Random-Forest classifier and then apply the trained model to new data — all without leaving the browser tab.

The application is served by a Django backend that handles file I/O and model training, while the frontend provides a real-time 3D viewport and interactive annotation tools.

📂 Load Data

Loading Data

How to load your point cloud dataset into the viewer.

Supported Formats

  • .ply — PLY Point Cloud
  • .glb — GLB Model
  • .las — LAS Point Cloud

How to Load a File

File -> Load Data -> Click to select file.

Loading Pipeline

The file will be converted in LAS -> Potree -> Viewer

ℹ️ Files larger than 50 M points may take several minutes to load and require at least 8 GB RAM.

🎯 Training Mode

Training Mode

Annotate point cloud data and train a Random Forest classifier.

Managing Classes

Selection and cut mode

Assign to class

Calculate Features

Calculate the features to use for the dataset

Divide the dataset

Segment the dataset into training and validation sets

Training configuration

Choose the two point cloud data, previously segmented

Choose which features to be used to train the RF model (also if include RGB)

Decide the values of the parameters to give in input to the RF model.

Those parameters are:

- n of jobs (default: 12)

- n of estimators (default: 200)

- max depths (default: 15)

- min samples split (default: 20)

- max features (default: "sqrt")

- use gpu (default: "off")

Training results

After training the model trained and report will be given as output.

The viewer will explain the report with sections:

- metrics accuracy

- metrics accuracy per class

- sorted feature importance

- different processing time for each steps (loading, training, computing metrics, and overall time)

Export / Download Dataset
